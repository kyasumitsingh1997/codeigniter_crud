Getting Start

This is simple Codeigniter 4 tutorial for beginner who want to learn basic Crud Mysql Database Operation in Codeigniter 4 framework

Requiements

  • PHP version 7.4 or newer is required, with the *intl* extension and *mbstring* extension installed.
  • The following PHP extensions should be enabled on your server: php-json, php-mysqlnd, php-xml

Installation

Just download the project in your computer and then upload script folder on server or localhost.

  1. Open phpmyadmin
  2. Create New Database named "test_crud".
  3. Import database tables in newly created database "test_crud" from downloaded folder -> test_sql -> test_crud.sql.zip.
  4. Open Project in a Code Editor, open the file database.php in PROJECT_FOLDER/app/config/ and modify database, username, password to your credentials.
  5. Proceed to : yourdomain.com or localhost/PROJECT_FOLDER
